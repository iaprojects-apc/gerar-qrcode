Passo 1

Como preparar seus vídeos

Importante: O QR Code armazena apenas um link — não o vídeo em si. Siga estes passos:
  1. Google Drive: Faça upload do vídeo → Compartilhar → "Qualquer pessoa com o link" → Copie o link.
  2. YouTube: Faça upload como Não listado → Copie o link do vídeo.
  3. WhatsApp/Google Fotos: Compartilhe o vídeo e copie o link gerado.

Depois, cole o link no formulário abaixo e gere o QR Code de cada mãe.
